## Detailed Description

```text
Amazon Warehouse Jobs UK helps paid users monitor supported Amazon UK hiring pages and continue the booking flow when a matching warehouse shift appears.

The extension is designed for users who already use Amazon Jobs. It runs only on supported Amazon hiring, authentication, and application pages. Users choose their local search settings, enter the buyer email and Amazon job-search email, and activate the extension after paid access is valid.

Paid access is checked through getslotnow.com. Checkout is handled by Dodo hosted checkout. The Amazon job-search email entered during checkout is the Amazon Jobs account allowed to use that access.

Key behavior:
- Runs on supported UK Amazon hiring pages.
- Lets the user configure location, job type, interval, and booking preferences.
- Blocks booking automation until paid access is valid.
- Supports 30-day access and annual/pro access.
- Keeps payment handling outside the extension through Dodo hosted checkout.

This extension is not affiliated with Amazon.
```
