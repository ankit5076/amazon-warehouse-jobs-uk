This extension works only on supported Amazon UK hiring job-search/application pages.

Test flow:
1. Install the extension.
2. Open the extension popup. The Activate toggle is user-controlled.
3. Configure the location scope, job type, and fetch interval.
4. Open a supported Amazon UK job-search page and sign in to Amazon manually if prompted.
5. Turn on Activate from the popup.
6. The extension polls supported Amazon UK pages and continues the existing Amazon UK booking/application flow when a matching shift appears.

This build does not include paid access, checkout, or payment-gateway calls. Payment integration is planned for a later tracker-service-backed release. The extension does not store an Amazon password or PIN.

This extension is not affiliated with Amazon.
